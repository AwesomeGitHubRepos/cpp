# ([C++](Cpp.md)) [gcov](CppGcov.md)

[gcov](CppGcov.md) is a GCC [tool](CppTool.md) to measure code coverage.

The webservice [Codecov](CppCodecov.md) can visualize the [gcov](CppGcov.md) results.
